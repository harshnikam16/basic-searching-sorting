# Bubble Sort Algorithm

## Algorithm
1. **Start**
2. Define an array
3. For each element `i` in the array:
   - For each element `j` in the array:
     - If `array[j] > array[j+1]`, then:
       - Swap `array[j]` and `array[j+1]`
4. **End**

## Pseudocode
```
START
    INPUT array 
    FOR i in array
        FOR j in array
            IF array[j] > array[j+1]
                SWAP array[j] and array[j+1]
    END FOR 
END
```
